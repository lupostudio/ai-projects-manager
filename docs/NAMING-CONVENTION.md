# Naming Convention

This document outlines the naming pattern used in the repository for consistency across all projects. The naming convention is structured as follows:

```
[category]-[project-name]-[ai-tool]
```

## Categories
1. **creative**: Projects that focus on creative aspects, such as artistic AI tools or creative writing.
   - **Example**: `creative-digital-painting-resnet`

2. **tool**: Utility projects or tools that enhance the functionality of AI applications.
   - **Example**: `tool-data-visualizer-d3`

3. **showcase**: Projects that demonstrate the capabilities of certain AI technologies with real-world applications.
   - **Example**: `showcase-image-classification-cnn`

4. **experiment**: Experimental projects aimed at testing new methodologies or techniques.
   - **Example**: `experiment-nlp-model-bert`

By following this naming convention, we ensure clarity and recognition across all projects within this repository.
